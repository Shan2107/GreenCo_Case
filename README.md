SAPP Market Data Analyzer & GreenCo Dashboard

This project provides interactive tools for analyzing Southern African Power Pool (SAPP) energy market data. It includes:

SAPP Market Data Analyzer:
A command-line tool to explore, filter, and summarize SAPP market data, including price and volume statistics, peak hour analysis, and CSV export.

GreenCo Dashboard Generator:
A script to generate visual dashboards for revenue at risk and price volatility exposure, using customizable PV generation profiles and market data.

Features:

Load and clean SAPP Excel data
Interactive menu for data exploration
Summary statistics and market comparisons
Visualizations for revenue and risk analysis
Export options for further reporting
Requirements:
Python 3, pandas, matplotlib, seaborn, numpy, openpyxl
